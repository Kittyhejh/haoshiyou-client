# Title

A project to help my San Francsco friends match landlords, tenants and roommates.

## Getting Start
```bash
# install and set up
npm install 
sudo npm -g cordova ionic
ionic state restore

# start web server
foreman start

# auto update, to perform after schema change
node haoshiyou-api/server/bin/autoupdate.js

```
